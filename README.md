# SourceScan

SourceScan是一个用于扫描源站点的工具，用于学习和研究目的。 必须在合规的审查期间才允许使用此工具，违规使用与作者无关。

## 安装

1. 克隆此仓库
2. 安装Python 3.x
3. 安装依赖项：`pip install -r requirements.txt`

## 使用

1. 进入项目目录
2. 配置```main.py```后运行命令：`python main.py`
3. 等待扫描完成
4. 查看报告：`ip_response.txt`
